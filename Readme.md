# PROJETO API REST PHP - FRAMEWORK SLIM 4

### CODIGOS DE RETORNO
- 2000 - Tudo certo
- 2001 - Tudo certo com restrições
- 4000 - Erro de Token
- 4001 - Falha ao executar api

### Baixar dependências composer
Execute o comando abaixo para baixar as dependências do projeto
- composer update
